Comment Jouer a Color Guesser

Color Guesser est un jeu de precision visuelle. Le but est de reproduire la couleur cible avec la meilleure exactitude possible en ajustant teinte, saturation et luminosite.

Mode Memoire

Observe la couleur cible pendant quelques secondes, puis reproduis-la une fois qu'elle disparait.

  • Tu ajustes les 3 curseurs HSL.
  • Le score depend de l'ecart entre ta couleur et la cible.
  • 5 manches par partie.

Mode Nom Couleur

Un nom de couleur apparait (par exemple Corail profond), puis tu dois reconstruire sa couleur reelle.

  • Difficulte Facile, Normal ou Expert.
  • Le mode est ideal pour apprendre les familles de couleurs.

Mode Code

La cible est donnee en code HEX, RGB ou HSL. A toi de retrouver la couleur correspondante.

  • Tu peux t'entrainer a lire les formats de code couleur.
  • Le format peut etre fixe ou automatique.

Mode En Ligne

Joue avec des amis dans un salon prive et comparez vos scores en direct.

  • Le host choisit le mode avant le lancement.
  • Classement en temps reel sur 5 manches.

Astuces Pour Faire Un Bon Score

  • Commence par la teinte, puis ajuste saturation et luminosite.
  • Evite les grands mouvements de curseur en fin d'ajustement.
  • En mode Memoire, note mentalement si la cible est plutot vive ou desaturee.
